# 🚀 Social Media Platform

A modern full stack social media application built using the MERN Stack.

---

## 📌 Features

✅ User Registration & Login  
✅ Create Posts  
✅ View All Posts  
✅ Like Posts  
✅ Comment on Posts  
✅ Delete Posts  
✅ REST API Integration  
✅ MongoDB Database  
✅ Responsive User Interface  

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JWT
- bcryptjs
